aplikasi kritik saran (CRUD) dengan IONIC

1. copy folder krasan ke htdocs
2. import database krasan.sql ke mysql
3. ganti ip pada file www->js->services sesuai ip kamu atau gunakan localhost
4. ketik "ionic build android" untuk membuild menjadi .apk (gunakan ip laptopmu pada no.3 dan pastikan terkoneksi dalam satu jaringan hanphone dan laptop)
5. install .apk hasil build pada android kamu
6. untuk menjalankan pada browser -> buka cmd lalu masuk ke directory folder KritikSaran lalu ketik ionic serve --lab pada cmd
